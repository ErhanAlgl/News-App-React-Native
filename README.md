A React Native News app with search and load more functionalities for Android devices using https://newsapi.org/ API as data source.



## INSTALLATION

- Clone Repository
- cd News-App-React-Native
- npm i
- npm start

## FUNCTIONALITY

- News search feature
- News sharing feature
